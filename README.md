# SneakerTown SPA

Современный интернет-магазин кроссовок на React, Redux Toolkit, Tailwind CSS и Firebase.

# Технологический стек

Frontend: React (Vite)
Стейт-менеджер: Redux Toolkit (RTK)
Маршрутизация: React Router DOM
Стилизация: Tailwind CSS
Бэкенд & Аутентификация: Firebase Auth
Запросы к API: Axios

## Запуск

```bash
npm install
npm run dev
```
